## Телегамм BOT для сервиса [art]Lab...
##### [art-lab-telegram-bot]

Телеграмм бот написан на pyton с использованием библиотеки python-telegram-bot==20.6
Зависимости указаны в файле requirements.txt

Для разворачивания проекта запустие скрипт **setup.sh**


### Доступный функционал:
- добавить категорию (фото + описание),
- добавить товар (фото + описание + цена).
